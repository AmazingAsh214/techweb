# Raspberry Pi

## Firmware

Update firmware:

```
rpi-firmware update
```

## OS

Delete pi user: `sudo deluser -remove-home pi`

## Apps

### Kodi

[Install Kodi](http://kodi.wiki/view/HOW-TO:Install_Kodi_on_Raspberry_Pi#Raspbian)

```
sudo apt-get update
sudo apt-get install kodi
```

### VLC

```
sudo apt-get install vlc
```
